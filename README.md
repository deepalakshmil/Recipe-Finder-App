Recipe Finder - a responsive React app to quickly search, browse, and view recipes with interactive cards and modals. Perfect for finding meal inspiration with a clean and modern UI.

Features:
- Search for recipes by name using the search bar.
- Displays recipe cards with:
- Image
- Recipe title
- Short instructions preview
- Click a recipe card to view full instructions in a modal.
- Animated loading skeletons while fetching recipes.
- Responsive design for mobile, tablet, and desktop.
- Smooth hover effects and animations for interactive UX.

Technologies Used:
- React + TypeScript
- Custom React hooks (useFetchRecipes) for API fetching
- CSS for responsive design, hover effects, and animations
- Browser fetch API for recipe data
- No external state management required
